# MJ文字情報ビューア

MJ文字情報一覧表（約6万文字）をブラウザで検索・比較できるWebアプリケーションです。

## 機能

- **文字検索**: 漢字を入力して異体字を一覧表示
- **MJ番号検索**: MJ文字図形名（例: MJ000001）で検索
- **読み検索**: ひらがな・カタカナで読みから検索
- **比較モード**: 2つの文字の属性を並べて比較
- **IVS対応**: 異体字セレクタ付き文字の検索・表示
- **検索履歴**: 最近検索した文字を保存

## 使い方

1. 検索ボックスに文字、MJ番号、または読みを入力
2. 比較モードを有効にすると2つの文字を比較可能
3. URLクエリパラメータ `?q=塚` や `?q=国&q2=國` で共有可能

## データ出典

[MJ文字情報一覧表 Ver.006.02](https://moji.or.jp/mojikiban/mjlist/)（IPA）
